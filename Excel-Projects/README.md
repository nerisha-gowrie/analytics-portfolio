# Excel Projects

This folder contains data analysis projects using Excel.

## Project

### Customer Bike Purchase Analysis
- Analysed customer demographics and lifestyle data to identify factors influencing bike purchases  
- Used pivot tables, charts, and calculated fields for analysis  

**Key Insights:**
- Middle-aged customers show the highest purchase rates  
- Shorter commute distances increase likelihood of purchase  
- Mid-to-high income groups demonstrate stronger purchasing behaviour, indicating income level as a key driver
