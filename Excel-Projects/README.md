# Excel Projects

This folder contains data analysis projects using Excel.

## Projects

### COVID-19 Data Exploration
- Analysed global COVID-19 data to evaluate infection rates, death rates, and vaccination trends  
- Used joins, window functions, CTEs, and views for advanced analysis  

**Key Insights:**
- Infection and death rates vary significantly across countries  
- Larger populations show higher total cases but different infection rates  
- Vaccination trends show steady increases over time  

---

### Nashville Housing Data Cleaning
- Cleaned and transformed raw housing data to improve quality and usability  
- Standardised date formats, handled missing values, split address fields, and removed duplicates  

**Outcome:**
- Improved data consistency and accuracy  
- Prepared dataset for reliable analysis and reporting
