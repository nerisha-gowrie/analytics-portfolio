# Power BI Projects

This folder contains Power BI dashboards and visualisation projects.

## Project

### Data Professionals Survey Dashboard
- Developed an interactive dashboard analysing job satisfaction, career entry challenges, and workplace priorities  
- Used DAX, KPIs, and slicers for dynamic data exploration  

**Key Insights:**
- Many respondents report difficulty entering the data industry  
- Salary is the most important factor when choosing a job  
- Satisfaction with management and career growth varies across respondents
